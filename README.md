# Roulette Simulator

A premium, high-fidelity Roulette simulation platform built with React, TypeScript, and Chart.js. Test your betting strategies across American, European, and French roulette variants with a visual, interactive interface.

## ğŸš€ Features

- **Interactive Visual Table**: Place bets by clicking directly on a virtual roulette table.
- **Incremental Betting**: Click multiple times to add chips; total amount is displayed on each chip.
- **Right-Click to Remove**: Easily decrease or remove bets with a right-click.
- **Multi-Bet Support**: Place multiple chips on different areas and simulate the combined outcome.
- **Multiple Variants**: 
  - **American**: 0 and 00 pockets.
  - **European**: Single 0 pocket.
  - **French**: Includes the *La Partage* rule for even-money bets.
- **Advanced Simulation**: Run up to thousands of iterations and visualize the equity curve.
- **Real-time Analytics**: Track win rate, total profit/loss, and maximum drawdown.
- **Premium Design**: Sleek dark mode aesthetic with gold accents and smooth animations.
